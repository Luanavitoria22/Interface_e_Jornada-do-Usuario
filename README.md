# Interface_e_Jornada-do-Usuario

Bem-vindo ao repositório do projeto de um **Aplicativo de Plano de Saúde**.

## 🎯 Objetivo do Projeto

Desenvolver uma interface funcional e centrada no usuário, garantindo facilidade no agendamento de consultas psicológicas por meio de diferentes fluxos de navegação.
O aplicativo tem como principal objetivo permitir que o usuário **agende consultas com psicólogos**, oferecendo uma experiência simples, intuitiva e acessível.

## 🚀 Funcionalidades

- 📱 Carteirinha digital  
- 🧪 Consulta de exames  
- 🏥 Rede credenciada  
- 💰 Solicitação de reembolsos  

## 📌 Requisitos Atendidos

- ✔️ Desenvolvido no Figma  
- ✔️ Wireframe (Baixa fidelidade)
- ✔️ Fluxo navegável  
- ✔️ Dois caminhos distintos para a mesma tarefa
